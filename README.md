# Mobile Price Range Prediction

This project predicts the price range of mobile phones based on their specifications using Machine Learning.

## Dataset
The dataset contains 2000 mobile phone records with features like:

- Battery Power
- RAM
- Pixel Resolution
- Internal Memory
- Number of Cores
- Camera Quality

Target variable:
price_range (0–3)

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- XGBoost

## Machine Learning Models

- Logistic Regression
- Random Forest
- XGBoost

## Results

Best Accuracy: **96%**
